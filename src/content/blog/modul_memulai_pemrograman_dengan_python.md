# Memulai Pemrograman dengan Python untuk Pemula üêç

Python adalah bahasa pemrograman yang sering dijuluki *ramah manusia*. Sintaksnya bersih, mudah dibaca, dan tidak membuat dahi berkerut sejak baris pertama. Karena itulah Python menjadi pilihan utama bagi pemula yang ingin menapaki dunia pemrograman, sekaligus andalan profesional di bidang data science, web, hingga kecerdasan buatan.

Artikel ini akan menjadi pintu masuk Anda untuk mulai belajar Python dari nol, secara bertahap dan praktis.

## Mengapa Memilih Python?

Python dikembangkan oleh Guido van Rossum dan dirilis pertama kali pada tahun 1991. Hingga hari ini, Python terus tumbuh dengan komunitas global yang aktif dan ribuan pustaka pendukung.

Beberapa alasan Python sangat populer:

- Sintaks sederhana dan mudah dipahami
- Cocok untuk pemula maupun profesional
- Dapat berjalan di berbagai sistem operasi
- Digunakan di banyak bidang teknologi

## Persiapan Lingkungan Pemrograman

Sebelum menulis kode, kita perlu menyiapkan Python di komputer.

### Instalasi Python

1. Kunjungi situs resmi Python di [https://www.python.org](https://www.python.org)
2. Unduh Python versi 3.x sesuai sistem operasi
3. Saat instalasi, centang opsi **Add Python to PATH**
4. Selesaikan proses instalasi

### Mengecek Instalasi

Buka Command Prompt atau Terminal, lalu jalankan:

```bash
python --version
```

Jika versi Python tampil, berarti instalasi berhasil.

## Program Python Pertama

Tradisi setiap bahasa pemrograman selalu dimulai dengan menyapa dunia.

```python
print("Hello, World!")
```

Kode di atas akan menampilkan teks **Hello, World!** di layar. Sederhana, tapi inilah langkah pertama Anda sebagai programmer Python.

## Variabel dan Tipe Data

Variabel digunakan untuk menyimpan data agar dapat diolah oleh program.

```python
nama = "Budi"
umur = 20
tinggi = 170.5
```

Python secara otomatis mengenali tipe data dari nilai yang diberikan.

### Tipe Data Dasar di Python

- **Integer (int)**: bilangan bulat
- **Float**: bilangan desimal
- **String (str)**: teks
- **Boolean (bool)**: `True` atau `False`

## Operator Dasar

Operator digunakan untuk melakukan operasi terhadap data.

### Operator Aritmatika

- `+` penjumlahan
- `-` pengurangan
- `*` perkalian
- `/` pembagian

Contoh penggunaan:

```python
a = 10
b = 5
print(a + b)
```

## Input dan Output

Program Python juga dapat berinteraksi dengan pengguna.

```python
nama = input("Masukkan nama: ")
print("Halo", nama)
```

Kode tersebut akan meminta pengguna memasukkan nama, lalu menampilkannya kembali.

## Latihan Mandiri

Agar pemahaman semakin kuat, cobalah latihan berikut:

1. Buat program untuk menampilkan biodata sederhana
2. Buat program untuk menghitung luas persegi panjang
3. Buat program yang menerima input umur dan menampilkan umur tersebut
